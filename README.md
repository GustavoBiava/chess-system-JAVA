# Projeto: Sistema Jogo de Xadrez

O projeto foi construído ao decorrer das aulas do curso <a href="https://www.udemy.com/course/java-curso-completo">Java COMPLETO Programação Orientada a Objetos + Projetos</a> presente na plataforma Udemy, sendo conduzido pelo Prof. Nelio Alves. O principal objetivo do projeto foi aplicar os conceitos apresentados até o dado momento do curso atráves da prática, mais especificamente por meio de um projeto.

No projeto foram aplicados tanto conceitos básicos do Java, quanto conceitos mais avançandos dentro de Orientação a Objetos para a contrução de uma partida de xadrez.

## Conceitos aplicados
- Implementação de modelo de domínio
- Estruturas condicionais/repetitivas
- Tipos enumerados
- Coleções
- Construtores
- Sobrecarga
- Herança
- Polimorfismo
- Encapsulamento
- Tratamento de exceções
- Métodos/Classes abastratas
